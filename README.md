## 紹介

こちらは Ruby on Rails を使用した API です。  
リクエストメソッドに応じて CRUD の処理に対応しています。

## 使用技術

- [Ruby](https://www.ruby-lang.org/ja/)
- [Rails](https://rubyonrails.org/)
- [SQLite](https://www.sqlite.org/index.html)

## 開発手順

Rails コマンドを使用しています

```bash
rails
rails server // APIサーバー起動
```
